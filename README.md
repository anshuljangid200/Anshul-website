# Anshul | Full Stack Developer Portfolio

A modern, responsive, and minimalist personal portfolio website built to showcase my projects, skills, experience, and professional presence online.

## 🚀 Built With

This project leverages a modern tech stack to deliver a fast, interactive, and beautiful user experience:

- **Framework:** React 18
- **Language:** TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **UI Components:** Shadcn UI & Radix UI
- **Animations:** Framer Motion
- **Icons:** Lucide React & React Icons
- **State/Data Management:** React Query
- **Routing:** React Router DOM

## 🌟 Features Included (What's Ready So Far)

The portfolio currently includes the following fully integrated sections and features:

- **Navigation (`<Navigation />`)**: Smooth and responsive header navigation menu.
- **Hero Section (`<HeroSection />`)**: An engaging introduction with my headline and role.
- **Social Dock (`<SocialDock />`)**: A cleanly designed MacOS-like dock or floating menu linking to my social profiles (GitHub, LinkedIn, Twitter, etc.).
- **Tech Stack (`<TechStackSection />`)**: Visual showcase of my technical skills, tools, and frameworks.
- **Projects (`<ProjectsSection />`)**: Display of my featured works, previous web projects, and case studies.
- **Experience (`<ExperienceSection />`)**: Professional work history, roles, and achievements.
- **Payments Integration Section (`<PaymentsSection />`)**: A dedicated section to demonstrate/process payments seamlessly.
- **Contact (`<ContactSection />`)**: A functional contact form or details for clients and collaborators to reach out.
- **Footer (`<Footer />`)**: Standard bottom-page information and links.

## 🛠️ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16.0 or higher recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/) or [pnpm](https://pnpm.io/) or [bun](https://bun.sh/)

### Installation

1. Clone the repository to your local machine:
   ```sh
   git clone <your-repo-url>
   ```

2. Navigate into the project directory:
   ```sh
   cd anshul\ port
   ```

3. Install all dependencies:
   ```sh
   npm install
   # or
   bun install
   ```

### Running Locally

To start the local development server:

```sh
npm run dev
# or
bun run dev
```

Your app should now be running on `http://localhost:5173/`.

### Building for Production

To create a production-ready build:

```sh
npm run build
# or
bun run build
```

The compiled files will be output to the `dist` folder.
